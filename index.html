[News] Ново Правило
by Admin [18-12-2023]
27. Не натискайте Switch Character или Select Server по време на дуел
Не натискайте Switch Character, Select Server или Изход от играта по време на дуел, защото ще ви даде фал.
При 2 фал-а опонента ви ще получи точка.
Тези фал-ове се броят и за дуела, което значи че ако дадете Switch Character по време на дуел и после ударите опонента си преди старта, опонента ви ще спечели точка.
Това става въпрос ако дадете Switch Character след като играта ви пита дали сте Ready.

Ако дадете Switch Character или Select Server по време на самият дуел, когато трябва да се биете, директно ще даде точка на противника ви, вместо фал.

Ако ви дисконектне по време на дуел, също ще ви даде фал или точка за противника ви.
Админите не махат фал-ове или точки, ако ви е дисконектнало, за това не ги питайте, защото няма как да се разбере дали нарочно не сте излезнали или е имало проблем с интернета ви.
